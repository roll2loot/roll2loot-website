# Roll2Loot - Starter (Next.js + Supabase + Stripe)

This is a minimal starter to run Roll2Loot subscription site. Follow the `.env.example` and README steps in the project root to configure Supabase and Stripe, then deploy to Vercel.

This starter includes:
- Next.js app directory pages
- TailwindCSS basic setup
- Stripe checkout + webhook endpoints
- Supabase client + SQL for tables
- Cron / email endpoint to send digital crates

See `.env.example` for variables.
